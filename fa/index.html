میرا

چهارچوب مدیریت یکپارچه محتوا برای ساختن وبسایت‌های چندگانه

معرفی

میرا یک چارچوب یکپارچه مدیریت محتوا جهت ساختن وب‌سایتهای مجزا و چنگانه است

با استفاده از میرا میتوانید همزمان و در کنار هم وبلاگ، فوتوبلاگ، گالری عکس، پادکست، کتاب، رزومه، معرفی‌کارها، صفحات پروژه، صفحات راهنما و ... را در طبقات ساختمان میرا مدیریت کنید. هر طبقه یک وب‌سایت مستقل است که با سایر طبقات در تعامل می‌باشد و می‌تواند تنظیم، قالب و اجزای خود را داشته باشد یا از خصوصیات طبقه‌ی مرکزی پیروی کند.

ساده

برای استفاده از میرا نیاز به نصب هیچ برنامه‌ی اضاف مانند دیتابیس‌ها ندارید. تنها و تنها محتوایی که مینویسید و یک قالب برای ساختن خروجی وب‌سایت. میرا تمام کارهای باقی‌مانده را برای شما انجام خواهد داد

مدرن

بدنه‌ی محتوایتان را به هر شکلی که دوست دارید بنویسید، MarkDown, Textile, BBCode, HTML یا متن های ساده

قابل انعصاف

هیچ محدودیتی برای ساختن لیست‌های آرشیو مثل دسته بندی موضوعی، برچسب، فصل‌های کتاب و امثال این‌ها وجود ندارد، هر تعداد لیست که میخواهید بسازید. همچنین برای ساختن فیلدهای تکی مثل تیتر، تیتردوم، عکس، نام نویسنده و غیره هم محدودیتی دست و پایتان را نمیبندد، به هر رویایی که برای انتشار محتوایتان دارید ادامه دهید، میرا آن‌را عملی خواهد کرد

نصب

اگر از cpan minus استفاده میکنید

$ cpanm Mira

یا با استفاده از perl shell

$ perl -MCPAN -e shell > install Mira

آماده سازی طبقه اصلی

$ mkdir Your_Mira_Folder
$ cd Your_Mira_Folder
$ mira init

تنظیمات

در صورت نیاز فایل config.yml را ویرایش کنید

نوشتن اولین پست در طبقه‌ی وبلاگ

$ mira new --title="your first post title" --floor="blog"

ساختن سایت

$ mira build

پیش نمایش

$ mira view
